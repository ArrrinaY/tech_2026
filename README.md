# Dating Telegram Bot

## Этап 1 — Планирование и проектирование

- Описание сервисов
- Архитектура системы
- Схема базы данных
- Метрики и логирование (structlog + prometheus-client)

## Этап 2 — Разработка базовой функциональности 

- Подняты основные сервисы: `bot_service`, `profile_service`, `PostgreSQL`, `Redis`, `RabbitMQ`, `MinIO`
- Реализованы регистрация пользователя, заполнение/обновление анкеты и просмотр своей анкеты
- Подготовлен запуск через Docker Compose

## Этап 3 — Система анкет и ранжирования 

- Добавлен поиск анкет с лайками/пропусками и сохранением взаимодействий
- Реализована система ранжирования (первичный и поведенческий рейтинг)
- Добавлены системное меню команд Telegram и навигация кнопками в боте
- Реализованы ручной ввод имени, раздел/команда «Мои мэтчи» и выдача мэтчей из API
