# Task 1: PostgreSQL Transactions

Демонстрация трех сценариев транзакций:

1. Успешное оформление заказа (`COMMIT`).
2. Ошибка в заказе и полный `ROLLBACK`.
3. Перевод баланса c `SAVEPOINT` и частичным откатом.

## Запуск

```bash
docker compose up --build
```

После запуска контейнер `app` выполнит сценарии и выведет финальное состояние таблиц в лог.
