# Описание сервисов

1. **Bot Service** (aiogram 3.x)
   - Технология: aiogram 3.x + FastAPI (для внутренних эндпоинтов при необходимости)
   - Основные функции:
     - получение обновлений от Telegram
     - обработка команд /start, просмотр анкет, лайк/дизлайк, редактирование профиля
     - передача бизнес-логики в Profile Service и Rating Service
     - логирование всех входящих сообщений и ошибок
     - сбор базовых метрик (кол-во сообщений, кол-во ошибок, время ответа)

2. **Profile Service** (FastAPI + SQLAlchemy async)
   - Технология: FastAPI + SQLAlchemy async
   - Основные функции:
     - создание/редактирование/просмотр/удаление анкеты
     - хранение фотографий (пока в виде URL)
     - расчёт первичного рейтинга на основе:
       • полноты заполнения анкеты
       • количества фотографий
       • соответствия возрасту/полу/городу
     - логи + метрики операций с анкетами

3. **Rating Service** (FastAPI + SQLAlchemy async)
   - Технология: FastAPI + SQLAlchemy async
   - Основные функции:
     - хранение и расчёт трёх уровней рейтинга
     - Уровень 1 – первичный (по анкете)
     - Уровень 2 – поведенческий (лайки, мэтчи)
     - Уровень 3 – комбинированный (взвешенная сумма)
     - отдельная таблица ratings для удобного периодического пересчёта
     - логирование каждого пересчёта + метрики времени расчёта
